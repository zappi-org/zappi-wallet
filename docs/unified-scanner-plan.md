# Zappi POS 통합 스캐너 UX 리디자인

## 개요

현재 4개 ActionButtons (Receive, Send, Transfer, Analytics)를 3개 하단 네비게이션으로 교체:
- `[Transfer]` `[로고 버튼]` `[Analytics]`
- 로고 버튼 클릭 시 통합 스캐너 모달 열림

**레이아웃**:
```
┌─────────────────────────────────────┐
│                                     │
│  [Transfer]    [◉]    [Analytics]  │
│      ↔️       로고        📊        │
│              (크게)                 │
└─────────────────────────────────────┘
```

**역할**: 입력 감지 & 담당 화면으로 라우팅 (현재 화면을 그대로 사용하면 위험함)

**상태 머신**:
```
idle → scanning → detected → navigate-to-screen → close
  ↑                                                  │
  └────────────────── close ─────────────────────────┘
```

**UI 구조**:
```
┌─────────────────────────────────────┐
│ ← 스캔                              │
├─────────────────────────────────────┤
│                                     │
│   ┌─────────────────────────────┐  │
│   │                             │  │
│   │         📷 카메라            │  │ ← 항상 활성화
│   │         (스캔 영역)          │  │
│   │                             │  │
│   └─────────────────────────────┘  │
│                                     │
│   ┌─────────────────────────────┐  │
│   │ 주소, 인보이스, 토큰 입력...  │📋│ ← 텍스트 입력 + 붙여넣기
│   └─────────────────────────────┘  │
│                                     │
└─────────────────────────────────────┘
```

감지타입 : 순수 숫자, 라이트닝 인보이스, 라이트닝 주소, LNURL-pay, LNURL-withdraw, 토큰, casuh-request (추가로 가능한 것이 있다면 반영)
